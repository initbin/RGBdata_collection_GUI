# RGBdata_collection_GUI
实现图像上的RGB数据的采集
  1.打开一个文件图片显示在GUI界面上 
  2.在图片上需要采集的地方点击鼠标，获取RGB数据
  3.程序设定一次采集10/11个点
  4.采集后的数据存放在EXCEL中，存放的位置格式与文件名存在一定的关系（例：45-tt.jpg）
  5.标记后的图片进行保存（但分辨率下降了）
